Fiddler 设置自定义列规则内容 ​

环境信息 ​

当前环境：Windows10
Fiddler 版本：v5.0.20243.10853 for .NET 4.6.2

1. 打开 Fiddler 点击右侧 Fiddler Script ​

  • 添加代码（JScript .NET 的语言）

  • 另外一种打开 Fiddler Script 方式

2. 将设置代码在 class Handlers 中 ​

3. 点击保存代码 ​

4. 注意事项 ​

  • 保存 Fiddler Script 后 一定要重启 Fiddler

5. 具体代码拷贝 ​

  • Fiddler 设置展示列的请求耗时 代码
js
 public static BindUIColumn("请求耗时", 80)
    function CalcTimingCol(oS: Session){
        var sResult = "0";
        if ((oS.Timers.ServerDoneResponse > oS.Timers.ClientDoneRequest)) {
            sResult = (oS.Timers.ServerDoneResponse - oS.Timers.ClientDoneRequest).TotalMilliseconds.ToString("N0");
        }
        return sResult + "ms";
    }

public static BindUIColumn("开始时间", 80)
function BeginTimiinaCol(oS: Session){
	 return oS.Timers.ClientDoneRequest.ToString("HH:mm:ss:fff");
}

public static BindUIColumn("结束时间",80)
function EndTimingCol(oS: Session){
	return oS.Timers.ServerDoneResponse.ToString("HH:mm:ss:fff");
}

 public static BindUIColumn("请求方式",80)

function CalcMethodCol(oS: Session){
	if (null != oS.oRequest) return oS.oRequest.headers.HTTPMethod; else return String.Empty;
}
